How to Use the Application

  1. Navigate to the Dashboard
  2. Select the Powerline Detection tab
  3. Use the map to select a location by dragging the marker
  4. Upload images of the powerline area
  5. Click "Upload & Detect Powerlines"

  1. Navigate to the Dashboard
  2. Select the Tree Height tab
  3. Use the map to select a location
  4. Click "Get Tree Height"
  5. View the calculated tree height

  1. Navigate to the Dashboard
  2. Select the Fire Risk tab
  3. Use the map to select a location
  4. Click "Analyze Fire Risk"
  5. View the fire spread risk assessment
